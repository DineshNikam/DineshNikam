### hey, I'm Dinesh 👋

I've been building Unity games for **3 years** — mostly mobile, sometimes WebGL, occasionally things that
probably shouldn't exist (like a game prototype for Android Auto). I care a lot about **things feeling good**:
smooth frames, snappy UI, that satisfying pop when a button animates just right.

When I'm not shipping at work, I'm in a game jam, tinkering with shaders, or building editor tools nobody
asked me to build — but everyone ends up using.

location  : Ahmedabad, India  (open to relocate: Mumbai / Bengaluru / Pune)

side life : solo dev jams on itch.io  →  i-am-dinesh.itch.io
obsession : game feel, performance, and water shaders
status    : OPEN TO WORK

It's on my portfolio homepage and it genuinely makes me happy every time I look at it.
`Unity URP` `Shader Graph` `WebGL` `GPU impostors`

### what I actually care about

**🎯 Game feel over everything**
I'll spend two hours on a button animation that takes 0.2 seconds to play.
Juice is not optional. Juice is love.

**⚙️ Tools that disappear**
The best Unity editor tool is the one devs use without thinking about it.
I build those.

**📉 Performance as craft**
Getting 60 FPS on a phone that costs less than a AAA game is a puzzle I genuinely enjoy solving.
Unity Profiler is basically a game at this point.

**🌊 Shaders and real-time rendering**
Not an expert yet — but obsessed. Koi Pond's water was my gateway drug.
Currently learning more Shader Graph fundamentals.

**🏁 Game jams**
72 hours. One theme. No sleep. Ship something.
Solo dev jams are the best way I know to grow fast and have fun at the same time.
